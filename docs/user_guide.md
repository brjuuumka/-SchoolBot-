Руководство пользователя SchoolBot.

SchoolBot — это Telegram-бот, который заменяет бумажный дневник.

С его помощью вы можете:
- Смотреть оценки
- Узнавать расписание
- Получать домашние задания
- Видеть замечания учителей
- Получать уведомления о новых оценках и событиях

Как начать работу:
Шаг 1: Найти бота
В поиске Telegram введите название бота

Шаг 2: Запустить бота
Отправьте команду /start

Шаг 3: Войти в систему
Введите логин и пароль, которые выдал администратор школы

Роли пользователей:
Система поддерживает 5 ролей. Каждая роль имеет свой набор функций.
- Ученик - учащимся школы
- Родитель - родителям учеников
- Учитель - преподавателям
- Классный руководитель - классным руководителям
- Администратор - администрации школы


Инструкция для ученика:
Главное меню ученика
После входа вы увидите кнопки:
- Мои оценки
- Расписание
- Домашнее задание
- Замечания
- Сменить пользователя
  
Функции ученика:
- Мои оценки - показывает ваши оценки по всем предметам с указанием даты и учителя
- Расписание - показывает расписание уроков на сегодня, завтра или неделю
- Домашнее задание - показывает домашние задания на текущий день
- Замечания - показывает замечания от учителей (дата, предмет, текст)
- Сменить пользователя - выход из системы (можно войти под другой учетной записью)

Пример просмотра оценок:
Ваши оценки:
Математика: 5 (25.03.2026)
Учитель: Петрова
Русский язык: 4 (24.03.2026)
Учитель: Сидорова
Физика: 5 (23.03.2026)
Учитель: Кузнецов


Инструкция для родителя:
Главное меню родителя:
- Мой ребенок
- Стат недели
- Оценки ребенка
- Расписание
- Сменить пользователя

Функции родителя:
- Мой ребенок - показывает сводку по ребенку: ФИО, класс, средние баллы
- Стат недели - показывает отчет за неделю (оценки, пропуски, замечания)
- Оценки ребенка - показывает все оценки ребенка по предметам
- Расписание - показывает расписание класса ребенка
- Сменить пользователя - выход из системы

Уведомления для родителей:
Вы будете получать мгновенные уведомления, когда:
- Ребенок получил двойку
- Ребенок пропустил урок
- Учитель сделал замечание


Инструкция для учителя:
Главное меню учителя:
- Мои классы
- Домашнее задание
- Объявление
- Сменить пользователя

Как выставить оценку:
1. Нажмите "Мои классы"
2. Выберите класс (например, "10А")
3. Выберите ученика из списка
4. Нажмите "Поставить оценку"
5. Введите оценку (число от 1 до 5)
6. Подтвердите действие

Как отметить пропуск:
1. Выберите ученика
2. Нажмите "Отметить пропуск"
3. Подтвердите действие

Как отправить домашнее задание:
1. Нажмите "Домашнее задание"
2. Выберите класс
3. Введите текст домашнего задания
4. Нажмите Отправить


Инструкция для классного руководителя:
Дополнительные функции (в дополнение к функциям учителя):
- Мой класс - показывает список учеников класса с их средними баллами
- Родителям - отправляет сообщение всем родителям класса
- Ученикам - отправляет сообщение всем ученикам класса


Инструкция для администратора:
Главное меню администратора:
- Статистика
- Регистрация
- Рассылка
- Расписание
- Сменить пользователя

Функции администратора:
- Статистика - показывает общую статистику по школе (количество пользователей, успеваемость)
- Регистрация - создание новых учетных записей (ученик, родитель, учитель)
- Рассылка - отправка сообщения всем пользователям системы
- Расписание - управление расписанием уроков
- Сменить пользователя - выход из системы

Как зарегистрировать нового пользователя:
1. Нажмите "Регистрация"
2. Выберите роль: Ученик, Учитель, Родитель или Классный руководитель
3. Заполните данные: ФИО, логин (придумайте), пароль (сгенерируется автоматически)
4. Для родителя - укажите ребенка
5. Для учителя - укажите предмет
6. Для ученика - укажите класс


Уведомления:
Все пользователи получают push-уведомления в Telegram о важных событиях:
- Новая оценка - получают ученик и родитель
- Пропуск урока - получают ученик и родитель
- Новое замечание - получают ученик и родитель
- Новое домашнее задание - получает ученик
- Объявление - получают все получатели

Частые проблемы при использовании бота:
1. Что делать, если забыли пароль?
Обратитесь к администратору школы. Он может сбросить пароль и выдать новый.

2. Почему я не вижу оценки своего ребенка?
Убедитесь, что администратор привязал вас к ребенку при регистрации. Если нет - обратитесь к администратору.

3. Как посмотреть расписание на неделю?
Нажмите кнопку "Расписание", выберите свой класс. Бот покажет расписание на все дни недели.

4. Приходят ли уведомления, если я не в боте?
Да, уведомления приходят в Telegram в любое время, даже если бот закрыт.

5. Можно ли пользоваться ботом с компьютера?
Да, Telegram есть для Windows, Mac и Linux. Бот работает одинаково на всех платформах.

Безопасность:
- Ваши данные хранятся в зашифрованном виде
- Родитель видит только своего ребенка
- Учитель видит только свой класс
- Никто, кроме вас, не может войти в ваш аккаунт
